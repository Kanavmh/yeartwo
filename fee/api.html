Ajax

  • Dynamic Content Updates
  • Form submission without page reload
  • Auto completion and live search
  • Real time updates
  • Pagination
  • Real time chart updates

Client side: AJAX uses JS to send requests to the server

Server side: The server processes the request and sends data back in formats like XML, JSON, or HTML

Asynchronous nature: The browser doesn't wait for the response


Method Description Example
open(method, url, async) Initializes a request xhr.open('GET', 'https://api.example.com', true);
send(data) Sends the request to the server xhr.send();
setRequestHeader() Adds a label/value pair to the header to be sent xhr.setRequestHeader('Content-Type', 'application/json');
abort() Cancels the current request xhr.abort();
getAllResponseHeaders() Returns header information xhr.getAllResponseHeaders();
getResponseHeader() Returns specific header information xhr.getResponseHeader('Content-Type');
